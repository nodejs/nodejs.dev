## Un peu d'histoire

JavaScript est un langage de programmation qui a été créé par Netscape en tant qu'outil de script pour manipuler les pages web dans leur navigateur, [Netscape Navigator](https://en.wikipedia.org/wiki/Netscape_Navigator).

Une partie du modèle économique de Netscape consistait à vendre des serveurs Web, qui comprenaient un environnement appelé _Netscape LiveWire_ permettant de créer des pages dynamiques en utilisant JavaScript côté serveur. Malheureusement, _Netscape LiveWire_ n'a pas eu beaucoup de succès et le JavaScript côté serveur n'a été popularisé que récemment, avec l'introduction de Node.js.

Un facteur clé qui a conduit à l'essor de Node.js a été le timing. Quelques années auparavant, JavaScript avait commencé à être considéré comme un langage plus sérieux, grâce aux applications "Web 2.0" (telles que Flickr, Gmail, etc.) qui ont montré au monde ce que pouvait être une expérience moderne sur le Web.

Les moteurs JavaScript se sont également considérablement améliorés, car de nombreux navigateurs sont en concurrence pour offrir aux utilisateurs les meilleures performances. Les équipes de développement derrière les principaux navigateurs ont travaillé dur pour offrir un meilleur support pour JavaScript et trouver des moyens de rendre JavaScript plus rapide. Le moteur que Node.js utilise sous le capot, V8 (également connu sous le nom de Chrome V8 pour être le moteur JavaScript open-source du projet Chromium), s'est considérablement amélioré grâce à cette concurrence.

Node.js a été construit au bon endroit et au bon moment, mais la chance n'est pas la seule raison de sa popularité actuelle. Il introduit un grand nombre de réflexions et d'approches innovantes pour le développement JavaScript côté serveur qui ont déjà aidé de nombreux développeurs.

## 2009

- Node.js est né
- La première forme de [npm](https://www.npmjs.com/) est créée.

## 2010

- [Express](https://expressjs.com/) est né
- [Socket.io](https://socket.io) est né

## 2011

- npm hits version 1.0
- Les grandes entreprises commencent à adopter Node.js : LinkedIn, Uber, etc.
- [hapi](https://hapi.dev) est né

## 2012

- L'adoption se poursuit très rapidement

## 2013

- Première grande plateforme de blogs utilisant Node.js : [Ghost](https://ghost.org/)
- [Koa](https://koajs.com/) est né

## 2014

- The Big Fork : [io.js](https://iojs.org/) est un fork majeur de Node.js, dont l'objectif est d'introduire le support ES6 et d'aller plus vite.

## 2015

- The [Node.js Foundation](https://foundation.nodejs.org/) is born
- IO.js est fusionné de nouveau dans Node.js
- npm introduit les modules privés
- Node.js 4 (versions 1, 2 et 3 jamais publiées)

## 2016

- The [leftpad incident](https://blog.npmjs.org/post/141577284765/kik-left-pad-and-npm)
- [Yarn](https://yarnpkg.com/en/) est né
- Node.js 6

## 2017

- npm se concentre davantage sur la sécurité
- Node.js 8
- HTTP/2
- V8 introduit Node.js dans sa suite de tests, faisant officiellement de Node.js une cible pour le moteur JS, en plus de Chrome.
- 3 milliards de téléchargements de npm chaque semaine
